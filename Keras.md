## **Why Keras?**
Keras是一個開放原始碼，高階深度學習程式庫，使用Python編寫，能夠運行在TensorFlow或Theano之上。  
Keras可以使用最少的程式碼，花費最少的時間，就可以建立深度學習模型，並進行訓練、評估準確率，進行預測。相對的如果使用Tensorflow，雖然可以完全控制各種深度學習模型的細節，但是需要更多程式碼，花費更多時間開發，才能達成。

## **Keras安裝**
首先確認已安裝了Numpy和Scipy  
叫出terminal  
#如果是python2.X  
$pip install keras  
#如果是python3.X  
$pip3 install keras
